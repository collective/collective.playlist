===================
collective.playlist
===================

Provides a playlist with tracks

Features
--------

- Viewlet with button to modal with playlist player. Uses jplayer. Newest Playlist gets linked.


Examples
--------

This add-on can be seen in action at the following sites:
http://www.bluechurch.ch
